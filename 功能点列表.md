# 天合接单平台 - 功能点列表

> 基于一期 MVP 产品梳理，按模块与页面组织

---

## 一、学员端

### 1.1 首页（Home.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.1.1 | 平台入口导航 | 首页、接单大厅、天梯榜单、账户与收益、我的项目 |
| 1.1.2 | 快捷入口 | 一键跳转接单大厅、我的项目等核心页面 |

### 1.2 接单大厅（ProjectHall.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.2.1 | 分类 Tab | 短剧 / AI漫剧 / 短视频 / 综艺 |
| 1.2.2 | 项目卡片展示 | 项目名称、标签、内容类别、难度、交付周期、集数、参与团队、接单模式、名额、状态 |
| 1.2.3 | 筛选 | 分类、题材标签、截止时间区间、状态 |
| 1.2.4 | 排序 | 推荐（置顶+最新）、单价、最新发布、截止时间 |
| 1.2.5 | 接单资格判断 | 状态 Active、并发未满、项目可接单时显示接单按钮 |
| 1.2.6 | 抢单 | 满足资格一键接单，生成订单并创建协同空间 |
| 1.2.7 | 报名 | 提交报名申请，进入待审核队列 |
| 1.2.8 | 不可接单提示 | 产能满 / 已截止 / 已满员 / 账号状态不可接单 |

### 1.3 项目详情（ProjectDetail.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.3.1 | 项目信息展示 | 名称、分类、标签、需求、单价、集数、截止时间、交付周期、接单模式 |
| 1.3.2 | 一键接单（抢单） | 满足资格直接接单 |
| 1.3.3 | 报名按钮（报名模式） | 提交报名，幂等（重复点击不重复报名） |
| 1.3.4 | 接单状态展示 | 已接单 / 报名待审核 / 报名已驳回 / 可接单 |

### 1.4 我的项目（MyProjects.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.4.1 | 订单列表 | 所有订单（进行中、已提交、待修改、已验收等） |
| 1.4.2 | 报名记录展示 | 报名待审核、报名已驳回，与订单合并展示 |
| 1.4.3 | 类型列 | 区分「订单」「报名待审核」「报名已驳回」 |
| 1.4.4 | 状态筛选 | 进行中、已提交、退回修改、审核通过、已验收、已关闭、报名待审核、报名已驳回 |
| 1.4.5 | 进入项目空间 | 订单可进入协同空间；报名待审/已驳回仅展示状态 |
| 1.4.6 | 接单/申请时间 | 订单显示接单时间，报名显示申请时间 |

### 1.5 项目协同空间（ProjectSpace.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.5.1 | 概览 | 订单状态、截止时间、成员、订单信息 |
| 1.5.2 | 素材 | 素材包列表与下载 |
| 1.5.3 | 交付 | 上传交付文件、版本列表（V1/V2…）、变更说明 |
| 1.5.4 | 修改意见 | 审核退回原因、修改点、时间线 |
| 1.5.5 | 沟通 | 图文沟通、成员列表 |
| 1.5.6 | 审核与验收 | 审核结论、验收记录（有权限时可见） |
| 1.5.7 | 多轮修改 | 退回后重新上传，版本留痕 |
| 1.5.8 | 申请验收 | 审核通过后可申请验收 |

### 1.6 账户与收益（Account.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.6.1 | 累计收益 | 总收益展示 |
| 1.6.2 | 可提现余额 | 未结算/待打款余额 |
| 1.6.3 | 已完成订单数 | 已验收订单统计 |
| 1.6.4 | 收益明细 | 订单号、项目名、金额、入账时间、状态、扣款、结算月份 |
| 1.6.5 | 收益与账本一致 | 与财务对账口径一致，扣款同步 |

### 1.7 天梯榜单（Leaderboard.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 1.7.1 | 总接单量榜 | 按累计接单数排行 |
| 1.7.2 | 总收益榜 | 按收益排行（公开展示） |
| 1.7.3 | 近期活跃度榜 | 近 7 天行为加权排行 |
| 1.7.4 | 榜单切换 | 三维度切换并排序 |
| 1.7.5 | 展示字段 | 排名、昵称、对应指标 |

## 二、后台管理

### 2.1 项目管理（ProjectManagement.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.1.1 | 项目列表（台账） | 项目ID、名称、分类、城市、项目状况、单价、集数、截止时间、名额/已接单数、状态、负责人、订单统计 |
| 2.1.2 | 筛选 | 状态、城市、项目状况、项目阶段、分类、接单模式、难度、参与团队、统筹、截止时间、搜索 |
| 2.1.3 | 新建项目 | 跳转 ProjectEdit 创建 |
| 2.1.4 | 编辑项目 | 跳转 ProjectEdit 编辑 |
| 2.1.5 | 发布/下架 | 影响学员端可见性 |
| 2.1.6 | 关闭/归档 | 项目状态流转 |
| 2.1.7 | 进入项目空间 | 从列表进入协同空间 |
| 2.1.8 | 报名审核 | 报名模式项目的报名列表，通过/驳回（驳回原因必填） |
| 2.1.9 | 导出 | 导出项目数据 |

### 2.2 项目编辑（ProjectEdit.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.2.1 | 基础信息 | 项目名称、分类、标签、内容类别、难度、参与团队、城市、项目状况（一级）、项目阶段（二级） |
| 2.2.2 | 成本与预算 | 预算金额、支出金额 |
| 2.2.3 | 交付与排期 | 交付周期、截止时间、集数 |
| 2.2.4 | 接单配置 | 接单模式（抢单/报名）、名额、报名审核策略、指定候选人 |
| 2.2.5 | 需求与素材 | 项目需求文本、素材说明、素材包 |
| 2.2.6 | 统筹人员 | 项目负责人选择 |
| 2.2.7 | 保存/发布 | 草稿保存、发布上线 |

### 2.3 学员管理（StudentManagement.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.3.1 | 入驻审核队列 | 待审核学员列表 |
| 2.3.2 | 通过/驳回 | 通过生成人才档案；驳回需填原因 |
| 2.3.3 | 人才库 | 学员ID、昵称、状态、技能标签、等级、并发占用/上限、接单量/准时率/通过率 |
| 2.3.4 | 编辑标签/等级 | 调整学员能力标签 |
| 2.3.5 | 状态调整 | 暂停、冻结、解冻 |
| 2.3.6 | 并发上限 | 调整个人并发上限 |
| 2.3.7 | 导出 | 导出学员数据 |

### 2.4 任务审核（ReviewQueue.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.4.1 | 审核队列 | 订单号、项目名、接单人、第几集、是否定稿、版本号、提交审核时间、等待时长、截止时间 |
| 2.4.2 | 排序 | 临期优先 + 等待时长 |
| 2.4.3 | 进入审核 | 跳转 ReviewDetail 执行审核 |

### 2.5 审核详情（ReviewDetail.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.5.1 | 交付版本列表 | 查看历史版本与修改记录 |
| 2.5.2 | AI 预审结果 | 展示预审风险等级/命中标签（参考） |
| 2.5.3 | 人工审核 | 通过 / 退回 |
| 2.5.4 | 退回必填 | 退回原因 + 修改点必填 |
| 2.5.5 | 审核结论同步 | 同步至协同空间，写入审计日志 |

### 2.6 财务对账（FinanceManagement.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.6.1 | 账本列表 | 按项目分组，项目名称、ID、金额统计、订单总数、验收时间、扣款、结算状态、结算月份 |
| 2.6.2 | 统计卡片 | 项目数、验收订单数、应付金额、扣款调整、已结算/未结算金额、已结算订单数 |
| 2.6.3 | 筛选 | 结算状态、验收时间区间 |
| 2.6.4 | 扣款调整（单笔订单） | 针对项目下每笔订单分别录入扣款金额，扣款原因必填 |
| 2.6.5 | 结算状态 | 未结算 / 已结算，可切换 |
| 2.6.6 | 订单详情 | 查看项目下订单列表，含扣款列 |
| 2.6.7 | 导出对账报表 | 导出汇总与明细 |
| 2.6.8 | 入账规则 | 验收后入账，扣款影响学员收益与榜单 |

### 2.7 数据看板（DataBoard.html）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.7.1 | 整体概览 KPI | 短剧总数量、总制作集数、待进集数 |
| 2.7.2 | 短剧整体情况 | 饼图：按状况+城市分布 |
| 2.7.3 | 短剧进度集数分布 | 柱状图：待进/已交付/在制中集数 |
| 2.7.4 | 短剧项目情况 | 看板：按状况+城市分列，项目卡片 |
| 2.7.5 | 项目完成进度 | 看板：按阶段（精剪中、一审等）分列 |
| 2.7.6 | 短剧具体完成进度 | 项目卡片网格，进度条、已交付/总集数 |
| 2.7.7 | 短剧进审时间 | 订单接单与截止时间表格、甘特图 |

### 2.8 榜单与数据（DashboardManagement.html，已隐藏）

| 序号 | 功能点 | 说明 |
|------|--------|------|
| 2.8.1 | 榜单规则配置 | 维度、周期、刷新策略、活跃度权重 |
| 2.8.2 | 平台核心数据看板 | 供需、效率、质量、收益、健康度 |

## 三、状态与流转

### 3.1 订单状态

| 状态 | 说明 |
|------|------|
| 待接单 | ToBeTaken |
| 已接单 | InProgress |
| 已提交 | Submitted |
| 退回修改 | NeedRevision |
| 审核通过 | Approved |
| 已验收 | Accepted（入账节点） |
| 已关闭 | Closed |

### 3.2 项目状态

| 状态 | 说明 |
|------|------|
| 草稿 | Draft |
| 已发布 | Published |
| 已满/已截止 | ClosedForTaking |
| 已关闭 | Closed |
| 已归档 | Archived |

### 3.3 学员状态

| 状态 | 说明 |
|------|------|
| 待入驻审核 | PendingApproval |
| 在训 | Training |
| 可接单 | Active |
| 暂停 | Paused |
| 冻结 | Frozen |
| 退出 | Exited |

### 3.4 报名状态

| 状态 | 说明 |
|------|------|
| 待审核 | Pending |
| 已通过 | Approved |
| 已驳回 | Rejected |
| 已取消 | Cancelled |
