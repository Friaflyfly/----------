## 1. 平台简介

天合接单平台是专为短剧、AI 漫剧等内容制作领域设计的**项目接单与协同交付平台**。学员可以在平台上：

- 浏览并接取剪辑项目
- 下载素材、上传成品、与团队协同
- 查看收益与榜单排名

---

## 2. 进入平台

打开浏览器，访问平台首页（`Home.html`）。

首页提供以下快捷入口：

| 入口 | 说明 |
|------|------|
| 接单大厅 | 浏览并接取项目 |
| 天梯榜单 | 查看全平台排行榜 |
| 账户与收益 | 查看个人收益 |
| 我的项目 | 管理订单与报名 |

点击顶部导航或首页卡片即可进入对应页面。

## 3. 接单大厅：寻找项目

> 入口：顶部导航 → **接单大厅**（`ProjectHall.html`）

### 3.1 浏览项目

接单大厅按内容类型分为四个 Tab：

- **短剧** / **AI漫剧** / **短视频** / **综艺**

点击对应 Tab 切换分类，每个项目卡片展示：

- 项目名称、内容标签、难度（精品 / 工厂）
- 交付周期、总集数、参与团队
- 接单模式（抢单 / 报名）
- 可用名额、截止时间、当前状态

### 3.2 筛选与排序

- **筛选**：可按题材标签（霸总/穿越/男频/女频）、截止时间区间、状态筛选项目。
- **排序**：支持推荐（置顶+最新）、单价、最新发布、截止时间排序。

### 3.3 接单资格

满足以下条件，接单按钮才会可用：

1. 账号状态为**可接单（Active）**
2. 当前进行中订单数**未达到并发上限**
3. 项目状态为**可接单**（未满员、未截止、未下架）

若不满足，系统会提示具体原因（产能已满 / 已截止 / 已满员 / 账号不可接单）。

## 4. 项目详情与接单

> 点击项目卡片上的「查看详情」进入（`ProjectDetail.html`）

### 4.1 查看项目信息

项目详情页展示：

- 项目名称、分类、内容标签
- 项目需求描述（脚本方向、制作要求等）
- 单价（元/集）、总集数、预计总金额
- 交付周期、截止时间
- 接单模式（抢单 / 报名）

### 4.2 抢单接单

如果项目为**抢单模式**，且满足接单资格：

1. 点击「**一键接单**」按钮
2. 系统弹窗确认，点击「确定」
3. 接单成功后自动创建订单和项目协同空间
4. 约 3 秒后自动跳转到**项目协同空间**

> ⚠️ 同一项目只能接一次，重复点击不会生成重复订单。

### 4.3 报名接单

如果项目为**报名模式**：

1. 点击「**我要报名**」按钮
2. 可填写备注（可选），点击「提交报名」
3. 系统提示「报名成功，请等待运营审核」
4. 报名状态会显示在按钮旁：**报名待审核**

报名后的状态变化：

| 状态 | 含义 |
|------|------|
| 报名待审核 | 已提交，等待运营审核 |
| 报名已通过 | 审核通过，订单已自动创建，可进入协同空间 |
| 报名已驳回 | 本次报名未通过，可查看驳回原因 |

## 5. 我的项目：查看订单与报名

> 入口：侧栏 → **我的项目**（`MyProjects.html`）

### 5.1 列表说明

「我的项目」整合展示**所有订单**和**报名记录**，通过「类型」列区分：

| 类型标签 | 含义 |
|----------|------|
| 🔵 订单 | 已成功接单，有对应协同空间 |
| 🟠 报名待审核 | 已报名，等待运营审核 |
| 🔴 报名已驳回 | 报名未通过 |

### 5.2 订单状态说明

| 状态 | 含义 |
|------|------|
| 进行中 | 已接单，正在制作 |
| 已提交 | 已上传交付，等待审核 |
| 退回修改 | 审核退回，需修改后重新提交 |
| 审核通过 | 交付已通过审核，可申请验收 |
| 已验收 | 甲方验收通过，收益已入账 |
| 已关闭 | 订单已关闭 |

### 5.3 筛选

点击顶部「状态」下拉框，可按状态筛选，快速找到需要处理的订单（如所有「退回修改」的订单）。

### 5.4 进入项目空间

订单行右侧点击「**项目空间**」按钮，即可进入该订单的协同工作空间。

> 报名待审核 / 已驳回的记录无法进入项目空间。

## 6. 项目协同空间：交付与沟通

> 入口：我的项目 → 点击「项目空间」按钮（`ProjectSpace.html`）

协同空间是每个订单的工作区，包含以下 Tab：

### 6.1 概览

- 查看订单状态、截止时间、空间成员
- 了解订单整体进度与里程碑

### 6.2 素材

- 查看运营上传的**素材包列表**
- 点击下载脚本、源视频、参考样片等素材文件
- 仅空间成员可下载

### 6.3 交付

**上传交付文件（核心操作）：**

1. 点击「**上传交付**」按钮
2. 选择本地文件（视频/压缩包等）
3. 填写「本次变更说明」（如：第1集初剪完成）
4. 点击「确认上传」
5. 上传成功后，订单状态自动变为**已提交**，进入审核队列

**版本管理：**

- 每次上传生成一个新版本（V1-1、V2-1……）
- 版本列表可查看所有历史版本
- 版本不可删除，留存全量记录

> ⚠️ 退回修改后，请重新上传修改后的文件，系统会生成新版本，旧版本保留。

### 6.4 修改意见

- 查看审核方/甲方发布的**修改意见时间线**
- 每条意见关联到具体版本，包含退回原因和修改点说明
- 按时间线顺序展示，支持追溯多轮修改历史

**处理退回修改的流程：**

1. 在「修改意见」Tab 查看具体修改要求
2. 按要求修改作品
3. 切换到「交付」Tab，重新上传修改后的文件
4. 系统状态回到「已提交」，重新进入审核

### 6.5 沟通

- 与项目成员进行**图文沟通**
- 所有沟通记录留存，不可删除
- 如有疑问可在此提问，运营/审核会及时回复

### 6.6 审核与验收

当订单进入「**审核通过**」状态后：

1. 此 Tab 会显示「申请验收」按钮
2. 点击后甲方/运营进行最终验收
3. 验收完成后订单状态变为「**已验收**」
4. 对应金额**自动入账**到你的账户收益

## 7. 账户与收益

> 入口：侧栏 → **账户与收益**（`Account.html`）

### 7.1 收益概览

页面顶部展示核心收益指标：

| 指标 | 说明 |
|------|------|
| 累计收益 | 历史全部已验收订单金额之和（扣款后） |
| 可提现余额 | 尚未结算/打款的余额 |
| 已完成订单数 | 已验收的订单总数 |

### 7.2 收益明细

下方表格展示每笔收益记录：

| 字段 | 说明 |
|------|------|
| 订单号 | 对应订单的唯一编号 |
| 项目名称 | 所属项目 |
| 金额 | 该订单应得金额 |
| 入账时间 | 验收时间即入账时间 |
| 扣款 | 财务录入的扣款金额（如有） |
| 结算状态 | 未结算 / 已结算 |
| 结算月份 | 财务线下打款后标记 |

### 7.3 结算说明

- **入账节点**：订单验收（Accepted）后自动计入收益
- **提现/打款**：线下由财务打款，打款后平台标记「已结算」
- **扣款**：如有扣款由财务录入，会同步影响你的收益显示

## 8. 天梯榜单

> 入口：侧栏 → **天梯榜单**（`Leaderboard.html`）

榜单展示全平台接单方的排名，共三个维度：

| 榜单 | 排行依据 |
|------|----------|
| 总接单量榜 | 累计接单成功订单数 |
| 总收益榜 | 累计已验收金额（扣款后） |
| 近期活跃度榜 | 近 7 天行为加权得分 |

活跃度计算方式（供参考）：
- 登录次数：20%
- 提交集数：20%
- 接单集数：20%
- 交付集数：30%

点击顶部 Tab 切换不同榜单，展示字段包括排名、昵称、对应指标数据。

## 9. 常见问题

**Q：为什么接单按钮是灰色/不可点击？**

A：满足以下任意一条时接单按钮不可用：
- 账号状态不是「可接单」（如在训中、已暂停）
- 当前进行中订单数已达到并发上限
- 项目名额已满
- 项目截止时间已过
- 项目已下架

请联系运营确认账号状态或等待名额释放。

---

**Q：我的报名提交了，但一直是「待审核」怎么办？**

A：报名需要运营人工审核，通常 1 个工作日内处理。如等待过长，可在沟通渠道联系运营询问。

---

**Q：交付上传失败或上传后状态没有变化怎么办？**

A：请检查文件格式（视频/压缩包等），或刷新页面后重试。如问题持续，联系技术支持。

---

**Q：审核退回了，修改后怎么重新提交？**

A：进入项目协同空间 → 「修改意见」Tab 查看修改要求 → 修改作品后 → 「交付」Tab 重新上传文件，填写变更说明后提交即可。

---

**Q：验收通过了，收益什么时候到账？**

A：验收通过后收益**立即入账**（状态变为「未结算」），实际打款由财务线下操作，打款后平台标记「已结算」，具体时间以财务公告为准。

---

**Q：榜单收益和我的实际入账金额不一致？**

A：榜单收益 = 已验收金额 - 扣款调整，若财务有录入扣款调整会影响榜单数据，详情可在「账户与收益」的收益明细中查看。
